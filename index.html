📊 สถิติของฉัน

ℹ️ เกี่ยวกับ VOLT QUIZ

คุณสมบัติเด่น

  • หมวดหมู่ย่อย — เลือกเรียนเฉพาะเรื่อง เช่น ตัวเก็บประจุ ไฟ AC แม่เหล็กไฟฟ้า ความปลอดภัย ฯลฯ
  • 10 ระดับความยาก — ปรับจากพื้นฐานถึงผู้เชี่ยวชาญ ระบบจะคัดคำถามให้เหมาะกับระดับ
  • สุ่มชุดคำถามใหม่ — กดรีเฟรชเพื่อสลับคำถามชุดใหม่ได้ตลอด
  • เฉลยละเอียด 3 ส่วน — สูตร 📐 + วิธีคิดทีละขั้น 🧮 + เกร็ดความรู้ 💡
  • 2 โหมด — "ฝึกฝน" เฉลยทันทีหลังตอบ / "สอบจริง" เฉลยรวมตอนจบ
  • จับเวลา · เสียงเอฟเฟกต์ · โหมดสว่าง/มืด และบันทึกคะแนนสูงสุดอัตโนมัติ

เคล็ดลับการใช้งาน

  • บนคอมพิวเตอร์ กดเลข 1–4 เพื่อเลือกคำตอบ และ Enter เพื่อไปข้อถัดไป
  • เลือก "ทุกหมวด" + ระดับผสม เพื่อทำข้อสอบรวมแบบจับฉลาก
  • คะแนนจะถูกบันทึกในเบราว์เซอร์นี้เท่านั้น ดูได้ที่เมนู "สถิติของฉัน"

🤖 ผู้ช่วย AI (Google Gemini)

กดปุ่ม ถาม AI ใต้เฉลยแต่ละข้อ (หรือเมนู "ผู้ช่วย AI") เพื่อให้ Gemini อธิบายเพิ่มเติม ขอตัวอย่าง หรือไขข้อสงสัยของโจทย์ข้อนั้นๆ

  • ขอ API key ฟรี ที่ aistudio.google.com/apikey แล้วกดไอคอน ⚙️ ในหน้าต่าง AI → เลือก "API Key" → วางคีย์ลงไป
  • เลือกโมเดลได้ เช่น gemini-2.5-flash (ค่าเริ่มต้น), gemini-2.5-flash-lite (เร็ว/ประหยัด), gemini-3.5-flash (ฉลาดสุด)
  • แสดงสมการสวยงาม คำตอบที่มี LaTeX (เช่น $V=IR$ หรือ $$P=I^2R$$) จะถูกแปลงเป็นสัญลักษณ์คณิตศาสตร์ด้วย MathJax อัตโนมัติ
  • โหมด Proxy — สำหรับเว็บสาธารณะ ตั้งเซิร์ฟเวอร์ตัวกลาง (Cloudflare Worker/Vercel) ที่เก็บคีย์ฝั่งเซิร์ฟเวอร์ แล้ว forward ไป generativelanguage.googleapis.com คีย์จะไม่รั่ว
  • ⚠️ อย่าใส่ API key ลงในไฟล์ HTML ที่ push ขึ้น GitHub เพราะจะถูกเปิดเผยต่อสาธารณะ — ใส่ผ่านหน้าตั้งค่าตอนใช้งานเท่านั้น (หรือใช้ Proxy)
